Night Fury Radar is a Next.js-based sonar exploration game. Follow these steps to get up and running.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** 16.x or higher
- **npm** (comes with Node.js)

<Note>
  This project uses Next.js 16.1.6 with React 19.2.3 and React Three Fiber for 3D rendering.
</Note>

## Installation

<Steps>
  <Step title="Clone the repository">
    Clone the Night Fury Radar repository to your local machine:

    ```bash
    git clone <repository-url>
    cd night-fury-radar
    ```
  </Step>

  <Step title="Install dependencies">
    Install all required packages using npm:

    ```bash
    npm install
    ```

    This will install:
    - `next` (16.1.6) - The React framework
    - `react` and `react-dom` (19.2.3) - React libraries
    - `@react-three/fiber` (^9.5.0) - React renderer for Three.js
    - `@react-three/drei` (^10.7.7) - Useful helpers for React Three Fiber
    - `three` (^0.182.0) - 3D library
  </Step>

  <Step title="Start the development server">
    Launch the development server:

    ```bash
    npm run dev
    ```

    The server will start on `http://localhost:3000`
  </Step>

  <Step title="Open in your browser">
    Navigate to [http://localhost:3000](http://localhost:3000) to see the game.

    You should see the Night Fury Radar start screen with the title and controls.
  </Step>
</Steps>

## First Play

Once the game loads, you'll see the start screen with control instructions.

### Game Controls

<CodeGroup>
```text Movement
[W A S D]  or  [↑ ↓ ← →]  —  FLY
  W/↑  - Fly up
  S/↓  - Fly down  
  A/←  - Fly left
  D/→  - Fly right
```

```text Actions
[SPACE]  —  SONAR PULSE
Fire a sonar pulse to reveal obstacles

[ESC]  —  PAUSE
Pause the game and access menu
```
</CodeGroup>

### How to Play

1. **Press SPACE** on the start screen to begin
2. **Navigate** using WASD or arrow keys to avoid obstacles
3. **Fire sonar pulses** with SPACE to reveal hidden obstacles in the dark
4. **Collect powerups** for special abilities:
   - **◆ Immunity** - Temporary invulnerability (green)
   - **◉ Fireball** - Destroy obstacles with your next pulse (red)
   - **◈ Plasma** - Instantly restore pulse energy (blue)

<Tip>
  Watch your plasma energy bar in the top right - you need energy to fire sonar pulses. It recharges automatically, faster when you're moving quickly.
</Tip>

### Game Mechanics

The game is located in `app/components/NightFuryRadar.js` and features:

- **Canvas-based rendering** - Full-screen HTML5 canvas with 2D context
- **Dynamic obstacles** - Procedurally generated pillars with varying heights and gaps
- **Sonar system** - Fire pulses to temporarily reveal obstacles (lines 232-274)
- **Powerup system** - Three types of collectible powerups (lines 69-80)
- **Score tracking** - Distance-based scoring with high score persistence

<Warning>
  The game difficulty increases as you move forward - scroll speed scales with your position (line 377), and obstacles become more frequent and challenging.
</Warning>

## Next Steps

Now that you have the game running:

- Explore the **game mechanics** in `app/components/NightFuryRadar.js`
- Customize **obstacles** by modifying the `makePillar` function (lines 132-169)
- Adjust **difficulty** by changing constants like `GAP` (line 62) and `baseSpeed` (line 40)
- Add **new powerup types** to the `POWERUP_TYPES` object (lines 76-80)

<Tip>
  Try beating the high score! The game tracks your best distance and displays it on the start screen.
