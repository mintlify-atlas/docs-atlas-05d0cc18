    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get the game running locally in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Night Fury Radar repository to your local machine:
      
      ```bash
      git clone https://github.com/ohhmkar/code-it-ralph-26.git
      cd code-it-ralph-26
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install the required packages using your preferred package manager:
      
      ```bash npm
      npm install
      ```
      
      The game uses Next.js 16, React 19, and Canvas 2D API for rendering.
    </Step>
    
    <Step title="Start the development server">
      Launch the game in development mode:
      
      ```bash
      npm run dev
      ```
      
      Open [http://localhost:3000](http://localhost:3000) in your browser to play. The game runs entirely in the browser using Canvas API.
    </Step>
    
    <Step title="Play the game">
      Press **Space** to start flying and fire sonar pulses. Use **WASD** or arrow keys to navigate. The darkness is intentional — pulse frequently to reveal obstacles!
      
      <Tip>Your first few runs will likely end in crashes. That's normal! The game rewards patience and rhythm as you learn to "see" with sound.</Tip>
    </Step>
  </Steps>
</div>
