## Overview

This page documents all configuration constants and tunable parameters in Night Fury Radar. Values are extracted directly from the source code.

---

## Difficulty Settings

Core parameters that define gameplay difficulty.

<ParamField path="player.baseSpeed" type="number" default="4.5">
  Base world scrolling speed in pixels per frame. **Harder difficulty setting** (was 3.2 in easier version).
</ParamField>

<ParamField path="GAP" type="number" default="220">
  Minimum horizontal distance between obstacle pillars. **Harder difficulty setting** (was 270 in easier version).
</ParamField>

<ParamField path="obstacleGapVariance" type="number" default="80">
  Random additional spacing added to GAP: `GAP + Math.random() * 80`. Results in 220-300px spacing.
</ParamField>

<ParamField path="minObstacleHeight" type="number" default="60">
  Minimum height for obstacle pillars. **Harder setting** - taller minimum obstacles.
</ParamField>

<ParamField path="pairGapMultiplier" type="number" default="3.8">
  Gap calculator for paired obstacles: `H - pairH - player.height * 3.8`. **Harder setting** (was 4.5 for wider gaps).
</ParamField>

<ParamField path="pairProbability" type="number" default="0.6">
  Probability of generating paired top+bottom obstacles:
  - Initial obstacles: 60% (check `Math.random() > 0.4`)
  - Runtime obstacles: 75% (check `Math.random() > 0.25`)
  
  **Harder setting** (was 40% initial).
</ParamField>

<ParamField path="speedScaler" type="number" default="3.5">
  Speed multiplier based on player X position: `1 + (player.x / W) * 3.5`. **Harder setting** for faster scaling.
</ParamField>

## Player Physics

Movement and physics parameters.

<ParamField path="player.gravity" type="number" default="0.13">
  Constant downward acceleration per frame.
</ParamField>

<ParamField path="player.lift" type="number" default="-0.48">
  Upward thrust when pressing up key (negative = upward).
</ParamField>

<ParamField path="player.downLiftMultiplier" type="number" default="0.7">
  Down key applies `player.lift * 0.7` for gentler descent control.
</ParamField>

<ParamField path="player.drag" type="number" default="0.96">
  Vertical velocity decay multiplier (< 1 creates air resistance).
</ParamField>

<ParamField path="player.dragX" type="number" default="0.93">
  Horizontal velocity decay multiplier.
</ParamField>

<ParamField path="player.rightAccel" type="number" default="0.22">
  Acceleration when pressing right key.
</ParamField>

<ParamField path="player.leftAccel" type="number" default="0.14">
  Acceleration when pressing left key (slower than right).
</ParamField>

<ParamField path="player.minX" type="number" default="40">
  Minimum allowed X position (left boundary).
</ParamField>

<ParamField path="player.maxXRatio" type="number" default="0.65">
  Maximum X as ratio of width: `W * 0.65` (right boundary).
</ParamField>

<ParamField path="player.width" type="number" default="56">
  Collision box width.
</ParamField>

<ParamField path="player.height" type="number" default="32">
  Collision box height.
</ParamField>

<ParamField path="player.pitchMultiplier" type="number" default="0.08">
  Visual rotation based on velocity: `Math.max(-0.4, Math.min(0.4, player.vy * 0.08))`.
</ParamField>

<ParamField path="player.maxPitch" type="number" default="0.4">
  Maximum rotation angle in radians (both up and down).
</ParamField>

## Collision Detection

<ParamField path="collisionTestPoints" type="array">
  Four test points for polygon collision:
  - `{x: cx, y: player.y + 2}` - top center
  - `{x: cx, y: player.y + player.height - 2}` - bottom center
  - `{x: player.x + player.width - 2, y: cy}` - right center
  - `{x: player.x + 4, y: cy}` - left center (offset 4px)
</ParamField>

<ParamField path="collisionMargin" type="number" default="2">
  Inset from edges for collision points (except left which uses 4px).
</ParamField>

<ParamField path="screenBoundaryMargin" type="number" default="5">
  Pixels outside screen bounds before triggering crash: `player.y < -5` or `player.y + player.height > H + 5`.
</ParamField>

<ParamField path="obstacleRenderMargin" type="number" default="80">
  Offscreen distance for obstacle rendering culling.
</ParamField>

<ParamField path="obstacleCullMargin" type="number" default="150">
  Distance behind screen before obstacles are removed from array.
</ParamField>

</ParamField>

<ParamField path="initialObstacleDistance" type="number" default="W + 250">

</ParamField>
